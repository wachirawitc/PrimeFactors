# PrimeFactors
Prime Factors on http://cyber-dojo.org

Factorize a positive integer number into its prime factors.</br>
</br>
For example:</br>
</br>
	 2 -> [2]</br>
	 3 -> [3]</br>
	 4 -> [2,2]</br>
	 6 -> [2,3]</br>
	 9 -> [3,3]</br>
	12 -> [2,2,3]</br>
	15 -> [3,5]</br>
</br>
</br>
Ref algorithm http://www.msgarciamath.com/whole-numbers/prime-factorization/
